<h2>《Effective C#》 :books: </h2> 

> BillWagner, Wagner  著    机械工业出版社

```C#
1.始终使用属性 (property)，而不是可直接访问的 Data Member。
  Always Use Properties Instead of Accessible Data Members.

2.为你的常量选择 readonly 而不是 const。
 Prefer readonly to const.

3.Prefer the is or as Operators to Casts. 

4.Use Conditional Attributes Instead of # if. 

5.Always Provide To String(). 

6.Distinguish Between Value Types and Reference Types.

7.Prefer Immutable Atomic Value Types. 

8.Ensure that 0 Is a Valid State for Value Types.

9.Understand the Relationships Among Reference Equals(),static Item Equals(),instance Equals(),and operator==.

10.Understand the pitfalls of GetHashCode(). 

11.Prefer foreach Loops. 

12.NET Resource Management. 

13.Prefer Variable Initializers to Assignment Statements 

14.Initialize Static Class Members with Static Constructors. 

15.Utilize Constructor Chaining. 

16.Utilize using and try/finally for Resource Cleanup. 

17.Minimize Garbage. 

18.Minimize Boxing and Unboxing 

19.Implement the Standard Dispose Pattern.

20.Prefer Defining and Implementing Interfaces to Inheritance. 
```
